# Alpine MI website — organised build

Static website prepared for GitHub Pages and standard web hosting.

## Structure
- Root: main pages such as `index.html`, `about.html`, and `projects.html`
- `case-studies/`: individual case-study pages
- `sectors/`: individual sector pages
- `assets/css/`: shared styles
- `assets/js/`: shared JavaScript
- `assets/images/logos/`: brand and accreditation logos
- `assets/images/projects/`: project photography grouped by project
- `assets/images/general/`: remaining general site imagery

All internal links and asset paths have been updated for the new structure.
